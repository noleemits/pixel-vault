AI-powered image bank for WordPress. Auto-assigns branded images, prevents duplicates across client sites, and generates on demand.

== Description ==

PixelVault is an intelligent image assistant for WordPress. Instead of browsing generic stock photo libraries, PixelVault understands your business and automatically finds or generates the right image for every post.

**Features:**

* Auto-assign featured images based on post content
* Site-wide visual style profile (mood, colors, tone)
* Anti-duplicate system — no two client sites get the same image
* SEO-optimized filenames generated from post keywords
* On-demand image generation with AI
* Works with Gutenberg, Elementor, Divi, and any theme

== Installation ==

1. Upload the `pixelvault` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Go to Settings → PixelVault
4. Enter your API key and configure your business profile
